How it works

Each of the … words is represented as a 300-dimensional vector, pre-trained by Google on ~100 billion words of Google News (2013). The vectors live in docs/data/wordvecs25000.js as a single JSON object. We normalise them to unit length so that cosine similarity is just a dot product — which is why the JavaScript stays a handful of lines.

The 3-million-word full model is ~1.6 GB, too large for a static web page, so we ship the top 25 000 most frequent single-word tokens. If your word is missing, try a common-noun synonym.
